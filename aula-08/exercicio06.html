Aula 08 - Operadores (Parte 2)

  1. Operadores Relacionais
    • ">"
    • "<"
    • ">="
    • "<="
    • "=="
    • "!="

  2. Operadores de Identidade
    • "==": igual (valor)
    • "===": idêntico (valor e tipo)

  3. Operadores Lógicos
    • "!": negação
    • "&&": e/conjunção
    • "||": ou/disjunção

  4. Operador Ternário
    • "?", ":": teste ? verdadeiro : falso

  5. Precedência
    1. Operadores Aritiméticos => "()", "**", "/"...
    2. Operadores Relacionais => Da esquerda para direita
    3. "!"
    4. "&&"
    5. "||"
