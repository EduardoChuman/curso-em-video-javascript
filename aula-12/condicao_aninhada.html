Aula 12 - Parte 1 (Condição Aninhada)

Condições também existem na linguagem JavaScript para validar determinada parte do código e assim criar possibilidades de comandos no seu código.

Exemplo: Um sistema para o usuário informar a idade e com base nisso verificar se ele pode votar, se não pode ou se votar é obrigatório.
