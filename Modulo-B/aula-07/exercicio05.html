Aula 07 - Operadores (Parte 1)

  1. Operadores Aritméticos
    • "+" - Soma: 5 + 2 => 7
    • "-" - Subtração: 5-2 => 3
    • "*" - Multiplicação: 5 * 2 => 10
    • "/" - Divisão: 5 / 2 => 2.5
    • "%" - Resto da Divisão: 5 % 2 => 1
    • "**" - Exponenciação: 5 ** 2 => 25

    • Ordem de Precedência (execução) dos operadores aritméticos:
      1. "()"
      2. "**"
      3. "*", "/" e "%"
      4. "+" e "-"

  2. Operadores de Auto-atribuição de Variável
    • n = n + 4 || n += 4
    • n = n - 4 || n -= 4
    • n = n * 4 || n *= 4
    • n = n / 4 || n /= 4
    • n = n % 4 || n %= 4
    • n = n ** 4 || n **= 4

  3. Operadores de Incremento
    • Pós Incremento
      • n = n + 1 => n++
      • n = n - 1 => n--
    • Pré Incremento
      • n = 1 + n => ++n
      • n = 1 - n => --n
