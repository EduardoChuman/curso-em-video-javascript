Aula 16 - Funções

Funções são ações executadas assim que são chamadas ou em decorrencia de algum evento.

Uma função pode receber parâmetro e retornar um resultado.

Exercício: Criar uma função que pega uma lista de um array numérico e informa se o número é par ou ímpar.

Funções também podem passar parâmentros default caso não seja enviado

  • Exemplo:
    function soma (primeiroNumero = 0, segundoNumero = 0) {
          return primeiroNumero + segundoNumero
    }

No JavaScript funções também podem ser atribuidas em variáveis

  • Exemplo:
    var numeroMultiplicado = function (numero) {
          return numero * 2
    }

Outra possibilidades falando em funções é a recursividade, ou seja, a função pode chamar ela mesma no bloco de ações

Exercício: Criar uma função que pega uma lista o mesmo array numérico do primeiro execicio e calcula seu faturial.
